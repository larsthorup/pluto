Pluto
=====

Backbone sandbox

Prerequisites:

* install node.js

then

    npm install


From the browser
----------------

run all tests

    file://(path-to)/src/test/index.html

run app

    http://localhost:8080/


From the command line
---------------------

continuously lint and test on every save

    grunt watch

perform static analysis

    grunt lint

run all tests

    grunt test

produce test coverage statistics (in output\coverage\out\coverage.html)

    grunt coverage

start server

    grunt server watch
